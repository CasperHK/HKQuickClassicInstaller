Ubuntu 速成輸入法安裝工具

View on GitHub Download .zip Download .tar.gz

介紹

  • 此工具提供一個簡易的速成輸入法安裝方案。

使用方法

  • 打開scripts文件夾，下載最新的安裝腳本，腳本名稱中v字以後的是版本編號，數值愈大愈新。
  • 進入腳本文檔。
  • 在 Raw 上按右鍵，另存文檔。
  • 按 Ctrl + Alt + T 打開Terminal。

  • 鍵入sh，再把下載好的文檔拉進去，就像以下一樣，按 Enter 。

    casper@casper-desktop:~$ sh '/home/casper/Desktop/hkqcinstaller_v1.0 (1).sh'
  • 輸入你的用戶密碼，按 Enter 。
